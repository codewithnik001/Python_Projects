# PhonePe Data Analysis Case Study

## Project Overview
This project analyzes PhonePe transaction, user, device, and demographic data using Python and Jupyter Notebooks.

## Objectives
- Data Loading and Understanding
- Exploratory Data Analysis (EDA)
- Data Quality Checks
- Data Merging and Advanced Analysis
- Data Visualization
- Business Insights and Conclusions

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook
- VS Code
