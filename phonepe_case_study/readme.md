# 📊 PhonePe Data Analysis Case Study

## Project Overview

This project is a comprehensive Data Analytics case study based on PhonePe transaction, user, device, and demographic datasets. The objective was to perform end-to-end data analysis using Python and derive meaningful business insights through data exploration, quality checks, data merging, visualization, and trend analysis.

The project was completed using Jupyter Notebooks in VS Code and focuses on real-world analytical techniques commonly used by Data Analysts and Business Analysts.

## Task 2: Exploratory Data Analysis (EDA)

* Analyzed transaction trends over time

## Task 6: Insights and Business Recommendations

### Key Findings

* Digital transactions have shown significant growth over time.
* Higher population density areas generally generate higher transaction volumes.
* User adoption continues to increase across multiple states.
* Transaction patterns indicate strong growth in digital payment usage.

### Recommendations

* Focus customer acquisition efforts in states with low user-to-population ratios.
* Increase marketing campaigns in high-density regions.
* Improve engagement strategies in lower-performing districts.
* Leverage transaction trends for targeted business expansion.
